About This Project

üìä Our Mission

The PoW Tokenomics Tracker provides transparent, verifiable data about Proof-of-Work cryptocurrency tokenomics. We believe in showing raw data and letting users make their own judgments about fairness.

üéØ Core Philosophy

Show Data, Don't Hide It We display raw allocations, not synthetic "fairness scores" that can mask manipulation.
Verifiable Sources Every data point links to its source. No "trust me bro" claims.
Community-Driven Anyone can contribute via GitHub. All data is reviewed by the community.
Transparency First We flag disclosure gaps and unknown allocations. What's hidden matters.

üîç What We Track

  • Genesis Allocations: Who got tokens before public mining/staking?
  • Miner Parity: When will miners catch up to premined allocations?
  • Investor Details: Which VCs invested? At what price? When does it vest?
  • Mining Economics: What does it cost to mine vs. what insiders paid?
  • Transparency Gaps: What information is hidden or undisclosed?
  • Supply & Emission: Current supply, inflation rate, halving schedules

ü§ù How to Contribute

This project is 100% community-driven. Anyone can add or update project data:

  1. Visit our GitHub repository
  2. Copy the project template and fill in the data
  3. Submit a Pull Request with your sources
  4. Community reviews and merges verified data

Check out our Contributing Guide for detailed instructions.
